CMO Diagnostic Dashboard

Setup Guide for Email + Slack Notifications

Two quick steps to connect notifications so you get alerted whenever a CMO submits a diagnostic request. Takes about 5 minutes total.

Step 1: Formspree (Email Notifications)

1Go to formspree.io and sign up (free tier = 50 submissions/month)

2Click New Form and name it "CMO Diagnostics"

3Copy the endpoint URL. It looks like:

https://formspree.io/f/xabc1234

4Open cmo-diagnostic-dashboard.html in a text editor

5Find this line near the top of the script section:

var B2BF_FORMSPREE_URL = "https://formspree.io/f/mbdzlkyz";

6Replace YOUR_FORMSPREE_ENDPOINT_HERE with your actual endpoint URL

7Formspree will send emails to whatever address you registered with

What you'll receive: An email with the CMO's email address, company domain, which diagnostics they selected, and a timestamp.

Step 2: Slack Webhook (DM Notifications)

1Go to api.slack.com/apps and click Create New App then From scratch

2Name it "Diagnostic Alerts" and select your workspace

3Click Incoming Webhooks in the left sidebar and toggle it On

4Click Add New Webhook to Workspace

5Select your own DM (Slackbot / your name) as the channel, then click Allow

6Copy the Webhook URL. It looks like:

https://hooks.slack.com/services/T.../B.../xxx

7Open cmo-diagnostic-dashboard.html in a text editor

8Find this line:

var B2BF_SLACK_WEBHOOK_URL = "YOUR_SLACK_WEBHOOK_URL_HERE";

9Replace YOUR_SLACK_WEBHOOK_URL_HERE with your actual webhook URL

What you'll receive: A formatted Slack DM with the CMO's email, company domain, selected diagnostics, and timestamp.

Step 3: Host on Your Divi Site

Since your site runs on Divi/WordPress, you have a few options:

OPTION A: EASIEST

Custom HTML Module

1In WordPress, create a new Page (e.g., "Market Diagnostics")

2In the Divi Builder, add a Code Module

3Paste the entire contents of cmo-diagnostic-dashboard.html into the module

4Publish the page

5Optionally set the page template to "Blank Page" for a clean look

OPTION B

Upload as Standalone HTML

1Upload cmo-diagnostic-dashboard.html to your WordPress /wp-content/uploads/ directory via FTP or File Manager

2Link directly to: b2bfusiongroup.com/wp-content/uploads/cmo-diagnostic-dashboard.html

OPTION C

Subdirectory

1Upload the HTML file to a subdirectory like /diagnostics/

2Access at: b2bfusiongroup.com/diagnostics/

Testing

1Open the HTML file in your browser locally first

2Select a diagnostic, enter a test email and domain, then submit

3Verify you receive both the Formspree email and the Slack DM

4Once confirmed, deploy to your site

Quick Reference

Formspree free tier: 50 submissions/month (plenty for CMO outreach)

Slack webhook: Unlimited, no expiration

No backend needed: Everything runs client-side in the browser

CORS: Formspree and Slack webhooks both support cross-origin requests from any domain

Mobile responsive: The dashboard works on all screen sizes

Divi safe: All CSS classes are prefixed with b2bf- to avoid conflicts
